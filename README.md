# Sistema de Segmentación de Clientes para una PYME mediante Machine Learning

## Descripción
Proyecto aplicado del curso preparatorio para la Maestría en Ciencia de Datos de la UIDE.
Segmentación de clientes usando el algoritmo K-Means con Python.

## Video explicativo
https://youtu.be/K7oMnkq-79E

## Archivos del proyecto
- `segmentacion_clientes.ipynb` — Notebook principal con EDA, visualizaciones y modelo
- `db_integration.py` — Script de integración con base de datos SQLite
- `clientes_pyme.db` — Base de datos con los 200 clientes segmentados
- `visualizacion_eda.png` — Gráfico de análisis exploratorio
- `segmentacion_clusters.png` — Gráfico de clusters K-Means
- `Informe técnico final.pdf` — Informe técnico completo

## Herramientas utilizadas
- Python 3.13, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SQLite3
- Jupyter Notebook en VS Code
